# Casino Game Simulator
โปรแกรมนี้ถูกออกแบบในลักษณะ แยกส่วนทำงาน เพื่อรองรับการพัฒนาโดยสมาชิก 6 คน โดยแบ่งออกเป็น 6 โมดูลหลักที่ทำงานเชื่อมโยงกันภายใต้ระบบกระเป๋าเงินกลาง

1. Core System  ฟและ Data Management (ส่วนกลางและจัดการข้อมูล)
      - Main Hub UI: พัฒนาระบบเมนูหลักด้วย Text-mode ที่สามารถวนลูป (Main Loop) 
        เพื่อให้ผู้ใช้เลือกเข้า-ออกเกมต่างๆ ได้อย่างต่อเนื่องโดยไม่ต้องปิดโปรแกรม
      
      - Wallett System: สร้างระบบจัดการหน่วยเงินกลาง โดยใช้ Pass by Reference    
         ในการส่งค่าตัวแปรเงิน (Credits) ไปยังฟังชันเกมต่างๆ เพื่อให้การตัดยอดเงินและ
         การรับรางวัลเกิดขึ้นจริงในฐานข้อมูลเดียวกัน

      - Player Profile: ระบบรับค่าชื่อผู้เล่นและเก็บสถิติพื้นฐาน (จำนวนครั้งที่ชนะ/แพ้) เพื่อนำไปแสดง
        ผลสรุป

2.  Blackjack  Game)
     - Deck Management: ประยุกต์ใช้ Array ในการจำลองไพ่ 52 ใบ และสร้างอัลกอริทึมในการสุ่ม  
       ข้อมูล

     - Dealer AI: สร้างเงื่อนไขให้ฝั่งเจ้ามือ (Ai) ตัดสินใจจั่วไพ่เพิ่มอัตโนมัติหากแต้มยังไม่ถึงเกณฑ์
       ที่กำหนด

3. Slot Gam
      - Pattern Matching: พัฒนาระบบสุ่มสัญลักษณ์ 3 ช่อง และใช้อัลกอริทึมตรวจสอบความเหมือน
         ของข้อมูล เพื่อกำหนดรางวัลต่าง ๆ

      - กำหนดความน่าจะเป็นของแต่ละสัญลักษณ์ : กำหนดความน่าจะเป็นในการออกรางวัลแต่ละแบบ    
         ไม่ให้เท่ากัน เช่น รางวัล Jackpot มีโอกาสออกเพียง 1% เพื่อจำลองความเสี่ยงจริง

4. Hi-Lo Game
      - Multiple Betting Options: ออกแบบระบบรับค่าการเดิมพันที่หลากหลาย เช่น แทงสูง/ต่ำ, แทง   
        เลขตอง, หรือแทงผลรวมแต้ม ซึ่งต้องใช้ Switch-Case และ If-else จำนวนมากในการตรวจสอบเงื่อนไข

      - Dice Simulation: ใช้ฟังก์ชัน rand() จำลองการทอยลูกเต๋า 3 ลูก และคำนวณผลรวมทางคณิตศาสตร์เพื่อตัดสินผลแพ้ชนะ

6. Lotto/Bingo
      - Array Comparison : ระบบให้ผู้เล่นเลือกชุดตัวเลขที่ชอบเก็บลงใน Array และทำการสุ่มเลขรางวัลขึ้นมาเปรียบเทียบ

      - Search Algorithm: เขียน Loop เพื่อค้นหาว่าตัวเลขที่ผู้เล่นเลือก ตรงกับเลขรางวัลจำนวนกี่ตัว และคำนวณเงินรางวัลตามลำดับขั้น

7. ระบบร้านค้าและบันทึกผ
      - File I/O System: พัฒนาระบบบันทึกข้อมูล (Save/Load) โดยเขียนยอดเงินคงเหลือ
        และชื่อผู้เล่นลงไฟล์ .txt เพื่อให้สามารถกลับมาเล่นต่อได้

      - Shop: ระบบร้านค้าที่สามารถนำเครดิตที่เล่นได้ มาแลกซื้อ ฉายาเท่ๆเจ๋งๆ หรือ ไอเทมต่างๆ
